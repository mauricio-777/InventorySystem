=====IMPORTANTE=====

Pasos para ejecutar el programa:

1. Ingresar a la terminal

2. Ejecutar el comando "dotnet run"

Despues de hacer cambio hacer el commit de la siguiente  manera:

git commit -m "comentario agregado por NOMBRE v1"
